# Day 03 | Python Data Types

> > Numerical Data Types

Python supports **three primary numerical data types**:

* **Integer (`int`)** – Represents whole numbers without a decimal point (e.g., `10`, `-25`).
* **Float (`float`)** – Represents numbers with decimal values (e.g., `3.14`, `-0.5`).
* **Complex (`complex`)** – Represents numbers with a real and an imaginary part (e.g., `4+2j`).

> > Arithmetic Operations

Arithmetic operators are used to perform mathematical calculations.

* **+** – Addition
* **-** – Subtraction
* ***** – Multiplication
* **/** – Division
* **%** – Returns the remainder after division.
* **** – Raises a number to the power of another.
* **//** – Returns the quotient without the decimal part (floor division).

> > Type Conversion

Type conversion changes a value from one data type to another using built-in functions.

```python
int()
float()
complex()
```

> > Comparison Operators

Comparison operators compare two values and return either `True` or `False`.

* **==** – Equal to
* **!=** – Not equal to
* **>** – Greater than
* **<** – Less than
* **>=** – Greater than or equal to
* **<=** – Less than or equal to

> > Mathematical Functions

Python provides built-in and `math` module functions for performing common mathematical operations.

```python
abs()
round()
min()
max()
pow()
math.floor()
math.ceil()
math.gcd()
math.sqrt()
math.log()
```

> > Strings

A **string (`str`)** is a sequence of characters used to store and manipulate text. Strings can be created using single quotes (`''`), double quotes (`""`), or triple quotes (`''' '''` or `""" """`) for multi-line text.

> > String Operations

Common operations that can be performed on strings include:

* Concatenation (`+`)
* Repetition (`*`)
* Indexing
* Slicing
* Finding the length using `len()`

> > String Methods

Frequently used built-in string methods:

```python
upper()
lower()
title()
strip()
replace()
split()
join()
```

> > Numbers vs Strings

* **Numbers** are used for mathematical calculations.
* **Strings** are used to store textual data.
* `10` is an integer, whereas `"10"` is a string. Although they look similar, they belong to different data types.

> > Key Takeaway

Python provides multiple data types for storing different kinds of information. Understanding numerical data types, arithmetic operators, type conversion, comparison operators, mathematical functions, strings, and string methods forms the foundation of Python programming.
